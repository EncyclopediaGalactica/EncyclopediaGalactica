            \textbf{Definition of n-th Root}\\
            If $n$ is any positive integer, then the principal n-th root of $a$
            is defined as follows:\\
            $\sqrt[n]{a} \text{ means } b^n = a $\\
            If $n$ is even, then we must have $a \geq 0 \text{ and } b \geq 0$\\
