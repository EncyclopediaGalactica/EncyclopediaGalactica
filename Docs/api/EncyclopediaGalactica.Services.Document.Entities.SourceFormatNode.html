SourceFormatNode entity which is the atomic building block of a data structure. This data structure is a tree where the nodes contains additional information about the following:

  • the structure itself meaning child entities
  • type of the node meaning it can be UI form
  • processing details meaning for example patterns
  • validation meaning validation rules for UI form
  • data related information
The spine of this whole structure possibilities is the SourceFormatNode providing the structure which makes possible to hook up additional information to the nodes.
